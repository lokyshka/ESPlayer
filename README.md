# ESPlayer
ESPlayer — DIY mp3-player project on ESP32. At the moment, the project is under development: hardware and software thinking. To create such a player, you need:
  1. ESP32-WROVER(we need about 3 megabytes of RAM)
  2. battery(i advise to take a 2000 mAh battery)
  3. 6 buttons
  4. charging card
  5. SD-cardreader(without Logic Level Shifter)
  6. TFT-display(55mm × 45mm)
  7. DAC-decoder with AUX
  8. ...(please wait updates)
